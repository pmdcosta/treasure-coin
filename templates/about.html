About Treasure Coin

Treasure Coin is a treasure hunt platform that allows players to find and hide crypto currency.

Any player can create a new exploration game and print the treasure codes. Once these codes are hidden, any participating player can find the hidden treasures and reap the rewards.

Through integration with the Open Simple Token API, the game organizer can attach currency to the treasures.

Once a player finds the hidden treasure, they can claim both the glory and the branded token reward.
