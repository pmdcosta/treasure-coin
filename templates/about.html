About

  1. How it works

The goal of Treasure Coin is to allow players to create or participate in treasure hunting games. In order to make the games more exciting we use blockchain technology, through the OST platform, to keep track of events and promote participation by using cryptocurrency rewards.

When you create your account you are automatically supplied 1 treasure coin, and you can use it to start a new treasure hunting game. Each game can have several hidden treasures and you are required to pay 0.1 coins for each treasure you create.

When creating a treasure, you need to supply a general location where it will be hidden and also a hint to help hunters find it. The created treasures will have an associated QR code which only the game creater can see. Print and hide this QR code. After that the hunt is on, and treasure hunters will try and find your hidden treasure.

When they find one, they simply need to scan the code with their smartphones. If they were the first to do so, they will be set as the discoverer of that treasure and also be rewarded 0.1 coins.
